Mon année à Apitic

De septembre 2017 à 2018, j'ai effectué ma licence pro en alternance à Apitic. c'est une PME française située à Lannion, dans le domaine de la programmation informatique. Cette société développe et commercialise principalement des logiciels permettant d’accompagner les commerçants ou les artisans comme la web caisse. Une web-caisse est un logiciel de caisse full web, c'est-à-dire une caisse fonction avec un navigateur web, sans aucun autre logiciel installé. La web caisse va permettre de gérer les commandes, les livraisons, les inventaires,...

Lors de mon année à apitic j'ai effectué des correctifs de bugs, des améorations de fonction, création de nouvelle fonction. Aussi j'ai pu crée un intranet pour gérer les comptes d'une application de gestion coupon ou encore j'ai pu réalisée un nouveau projet permettant de donné la possibilité aux futurs clients de souscrire eux meme à aux offres et options d'Apitic.
Langage utilisé durant l'année : Jquery, PHP, Symfony 1 et 2, Laravel, Bootstrap, Angular, Ionic, Slim, Lumen...
Méthode de gestion de projet : Scrum

Web Reportage Retro Geek

En 2017, lors d'un projet d'étude j'ai crée un web reportage sur retro geek. Dans l’application l’utilisateur va plongé dans univers du Retro gaming où comme dans un jeu il ya devoir resoudre une mission. Il va se promener dans diffèrents univers qui correspondront à une partie du reportage. Chaque univers fera référence à un jeu précis. Le but de l’utilisateur est de récolter toutes les informations. A la fin de chaque partie du reportage, il obtiendra une clef qui lui permettra d'accéder à la fin du web reportage.

Projet Scolaire : Recherche de Tweet Politique

En 2018, lors de ma license pro j'ai effectué différents projets dont un application sur le thème de twitter. L'objectif était "permettre à l’aide d’une application Internet l’interrogation d’une base de données constituées de Tweets, précédemment collectés. (par Api Tweet) Une application possible est notamment la réutilisation d’information en politique, afin par exemple d’opposé des citations à un concurrent politique lors d’un débat télévisé ou encore de procéder à une analyse d’opinion sur un sujet donné.

Fonctionalité :

  • La recherche de tweets sur des critères precis
  • La gestion des utilisateurs
  • un Application Responsive
  • La gestion de diffèrents types de profil d'utilisateur que peut definir l'admintrateur
